FAQs

Can I use TrackHire without an account?

Yes. TrackHire stores your data in browser local storage, so it works without sign-up or backend services.

How do I import or export my data?

Use the Settings page to export JSON/CSV and restore from a JSON backup whenever you need.

Is the dashboard responsive?

Absolutely. The app supports mobile, tablet, and desktop layouts with responsive navigation and cards.
