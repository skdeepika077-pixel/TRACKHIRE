# TrackHire

TrackHire is a polished, offline-first internship and job application tracker built with HTML5, CSS3, and Vanilla JavaScript.

## Features

- Landing page with hero, features, how it works, testimonials, FAQ, and CTA
- Dashboard, Applications, Add Application, Application Details, Analytics, and Settings views
- Local Storage persistence so the app works offline after first load
- Light and dark themes with preference saved locally
- Search, filters, sorting, and responsive mobile/desktop application views
- JSON export/import, CSV export, backup, restore, and clear data
- Canvas-based analytics charts and activity timeline
- Command palette, keyboard shortcuts, toast notifications, and modal confirmations

## Files

- `index.html` — main app and landing page
- `styles.css` — design system, responsive layout, theme styles
- `app.js` — application logic, routing, storage, charts, UI interactions
- `404.html` — static fallback page for deployment
