# TrackHire

TrackHire is a modern internship and job application tracking platform designed to help students and job seekers organize and manage their placement journey efficiently.

The application allows users to track internships and job applications, monitor application status, manage interview schedules, and stay organized throughout the hiring process.

## Features

* Add and manage internship and job applications
* Track application status (Applied, Assessment, Interview, Rejected, Selected)
* Search and filter applications
* Dashboard with application statistics
* Responsive design for desktop, tablet, and mobile devices
* Clean and intuitive user interface
* Local storage support for data persistence
* Application details page
* Progress tracking across the recruitment process

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Purpose

TrackHire was created to simplify the internship and job application process by providing a centralized platform where students can manage opportunities, track progress, and stay organized during placement and recruitment seasons.
